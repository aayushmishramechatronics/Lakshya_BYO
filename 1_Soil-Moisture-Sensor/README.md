# Soil Moisture Sensor with Leds on Arduino
## Material
You need the following things for this tutorial.
- Arduino Uno
- Soil Moisture Sensor for Arduino (x1)
- RGB LEDs (x3)
- 220-Ohm-Resistors (x3)
- Jumper Wires (x9)
